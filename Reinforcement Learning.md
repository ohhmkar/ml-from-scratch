## Reinforcement Learning

Let’s say we’re training a robot or an AI game agent.

We don’t have labeled data (like supervised learning).
We can’t group anything (like unsupervised learning).
Instead:

1. The agent tries actions (e.g., move left/right).
2. It gets feedback in the form of a reward or penalty.
3. It learns by trial and error → just like humans learn from experience.

#### Q. What does the agent learn from in reinforcement learning?

```
Trial and error (rewards and penalties)
```

#### Q. Which of the following best describes reinforcement learning?

```
It receives a penalty and learns to avoid it
```

#### Q. What is the "action" in reinforcement learning?

```
The move or step the agent chooses to take
```
